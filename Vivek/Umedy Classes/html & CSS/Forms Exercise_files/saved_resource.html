                            Kenneth Baltazar Nunez • 3 years ago • edited

                            Hi Baltazaar
                            <div> tags are mostly used to apply styles and contain content and/ or apply a "class" or "id" that isn't readily available. The HTML 5 specification has introduced a number of new elements that can (and should) be used instead of <div>. Examples of these new elements include <article>, <aside>, <header>, and <footer>, as well as others.
                            Consequently, the <div> element should pretty much only be considered as an extension mechanism which is used when there isn't another suitable, pre-defined, HTML element available. See http://www.w3.org/TR/html5/... .(W3C) for further info if needed!
                            Regards
                            Ken
